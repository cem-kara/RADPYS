# Code of Conduct

## Our Pledge

We are committed to a respectful, inclusive, and harassment-free community.

## Our Standards

Examples of positive behavior:

- Being respectful and constructive
- Accepting feedback gracefully
- Focusing on what is best for the project

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Publishing private information without permission
- Any discriminatory language or behavior

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code,
issues, and other contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies in all project spaces, including issues,
pull requests, and discussions.
